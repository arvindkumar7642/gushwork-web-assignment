# 🚀 Gushwork Web Development Assignment

 Overview

This project is a **responsive web page** developed using **HTML, CSS, and JavaScript (Vanilla JS)** based on a Figma design.

The objective was to build a **pixel-perfect UI**, implement interactive features, and follow modern web development best practices without using any external frameworks or libraries.

---

## 🎯 Features

### 🔝 Sticky Header

* Appears when scrolling down
* Hides when scrolling up
* Smooth transition animations

### 🖼️ Image Carousel

* Interactive horizontal slider
* Next/Previous navigation buttons
* Smooth sliding transitions

### 🔍 Image Zoom Preview

* Hover over images to see zoom preview
* Dynamic and user-friendly interaction

### 📱 Responsive Design

* Fully responsive across:

  * Desktop 💻
  * Tablet 📱
  * Mobile 📲

---

## 🧑‍💻 Tech Stack

* **HTML5** – Semantic structure
* **CSS3** – Flexbox, responsive layout, modern UI styling
* **JavaScript** – DOM manipulation & event handling
